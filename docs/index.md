# FEAGI Documentation

FEAGI is an open-source neuromorphic AI engine for embodied intelligence, combining high-performance Rust components with Python tooling to build, train, and run brain-inspired control systems for robots and simulations. These docs are organized as role-based journeys so you can start at the level that matches your goal and quickly find the right workflows, interfaces, and reference material.

## Choose Your Journey

- **Journey 1: Brain Design and Visualization**  
  Design, configure, and inspect FEAGI brains and visual workflows.  
  Go to [Brain Design and Visualization](./brain-builder/index.md), then start with [What Is a FEAGI Brain](./brain-builder/what-is-a-feagi-brain.md).

- **Journey 2: SDK Integrator**  
  Connect hardware, simulators, and applications through SDKs, APIs, and MCP.  
  Go to [SDK Integrator](./sdk/index.md).

- **Journey 3: Core Contributor**  
  Understand architecture, set up development, run tests, and navigate design records.  
  Go to [Core Contributor](./contributing/index.md).
