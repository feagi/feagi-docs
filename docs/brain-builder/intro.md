# Brain Builder

This section is for researchers, roboticists, and AI practitioners who want to design,
train, and evolve FEAGI brains connected to physical or simulated embodiments.

## What you will learn

- How a FEAGI brain is structured (genome, cortical areas, connectivity)
- How to use Neurorobotics Studio to create and configure brains
- How to navigate and inspect a live brain using the Brain Visualizer
- How to connect an embodiment and run training cycles
- How to use the genome hub to share and load pre-built brains

## Where to start

If you are new to FEAGI, start with [Concepts: The Genome](./concepts/genome.md) to
understand the fundamental building block of every FEAGI brain. Then follow the
[Neurorobotics Studio quickstart](./nrs/quickstart.md) to spin up your first brain.

If you already understand the concepts and want to use the visual tooling, jump to
[Brain Visualizer: Getting Started](./visualizer/getting-started.md).
